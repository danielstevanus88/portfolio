Description

Exspira, awarded "Best Category: Sustainability" at the Davinci Hackathon 2024, is a web application designed to help users reduce food waste and manage their groceries more effectively. The app allows users to track food expiration dates, sends reminders to prioritize consumption based on freshness, and even suggests daily meal ideas. Powered by the Anthropic API (Claude), Exspira generates recipes using selected ingredients, estimates expiration dates, and provides tailored suggestions for reducing waste. Built with React.js for an interactive user experience and efficient filtering features, and Express.js for robust backend API handling, Exspira combines practicality with sustainability to promote smarter food management.

Key feature includes:

  • Food Management: Add, remove, and update users' items(i.e. foods)
  • Expiration Date Prediction: Predict the expiration date of the user's food items
  • Expiration Date Tracker: Track the expiration date of the user's food items
  • Daily Notification: Send daily notifications to remind users of their food items' expiration dates
  • Recipe Generator: Generate recipes based on the user's selected ingredients
  • Food Waste Reduction: Provide tailored suggestions(e.g. donate the food before expiration date) to help users reduce food waste
  • Item Search and Filter: Search and filter the user's food items based on various criteria
  • Grocery Recipe Scanner: Scan the user's grocery receipt to add items to the app
